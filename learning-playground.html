üéØ DOM Manipulation - Control the Page!

The DOM (Document Object Model)

The DOM is how JavaScript sees and interacts with HTML. Think of it as a tree of elements you can modify!

document.getElementById('myId')           // Get one element by ID
document.querySelector('.myClass')        // Get first element matching CSS selector
document.querySelectorAll('div')          // Get all elements matching selector
element.textContent = 'New text'          // Change text
element.innerHTML = '<b>Bold!</b>'        // Change HTML
element.style.color = 'red'               // Change CSS
element.classList.add('active')           // Add CSS class

üéÆ Events & User Interaction

Events = Things That Happen

Events are actions that occur in the browser (clicks, keypresses, mouse movements, etc.)

element.addEventListener('click', function() { });     // Best practice
element.onclick = function() { };                      // Alternative
<button onclick="myFunction()">                        // Inline (avoid)

Common Events:
- click, dblclick        (Mouse clicks)
- mouseover, mouseout    (Hover effects)
- keydown, keyup        (Keyboard)
- change, input         (Form fields)
- submit                (Forms)
- load, resize          (Window)
